"만들며 성장한다"

이름 하나에 두 가지 다짐을 담았습니다.

단순히 소프트웨어를 개발하는 팀이 아닙니다. Build It이라는 이름처럼, 우리는 LUMI라는 프로젝트를 쌓아 올리는 동시에 팀원 한 명 한 명이 개발자로서, 동료로서 매일 조금씩 성장해가고 있습니다. 코드 한 줄이 제품의 기둥이 되는 것처럼, 이 과정 하나하나가 우리 자신을 만들어가는 재료가 됩니다.

Build the Project LUMI라는 완성도 높은 로컬 AI 어시스턴트를 하나씩 함께 쌓아 올립니다. 기술적 도전을 두려워하지 않고 정면으로 맞서갑니다.
Build Ourselves 프로젝트를 진행하며 각자가 개발자로서 더 깊어지고, 팀으로서 더 단단해집니다. 결과물만큼 과정도, 성장도 소중합니다.

황휘

UI 제작 · 프론트엔드

LUMI 프로젝트의 팀장으로서 전반적인 개발 방향을 설정하고 팀을 이끌고 있습니다. 사용자 경험을 최우선으로 생각하며, 복잡한 AI 기술을 누구나 직관적으로 사용할 수 있는 인터페이스로 풀어내는 것을 목표로 합니다. 웹 프론트엔드 개발을 통해 LUMI의 얼굴이 되는 화면을 직접 설계하고 구현하며, 프로젝트의 시각적 정체성과 사용성을 책임지고 있습니다.

담당 업무

  • 프로젝트 기획 및 전체 개발 방향 설정
  • UI/UX 설계 및 사용자 인터랙션 디자인
  • 프론트엔드 개발 (HTML · CSS · JavaScript)
  • 공식 웹사이트 제작 및 유지 관리

기술 스택

HTML CSS JavaScript Figma Git

강진구

백엔드 · 프론트엔드

서버 아키텍처 설계부터 프론트엔드 연동까지 폭넓은 풀스택 개발을 담당합니다. 안정적이고 확장 가능한 서비스 인프라를 구축하여 LUMI의 기술적 기반을 다지는 역할을 합니다. 백엔드와 프론트엔드 사이의 데이터 흐름을 매끄럽게 연결하며, 사용자 요청이 AI 엔진까지 빠르고 안정적으로 전달될 수 있도록 전체 통신 구조를 설계하고 최적화합니다.

담당 업무

  • 백엔드 서버 아키텍처 설계 및 구현
  • REST API 개발 및 엔드포인트 관리
  • 프론트엔드-백엔드 통신 구조 연동
  • 서비스 안정성 모니터링 및 최적화

기술 스택

Python Node.js REST API JavaScript Git

조성민

백엔드 · DB 엔지니어링

데이터베이스 설계와 백엔드 로직 구현을 전담하며 LUMI의 데이터 흐름 전체를 책임집니다. 효율적인 데이터 구조를 설계하여 AI 모델과 사용자 데이터 간의 원활한 연동을 가능하게 하며, 대용량 데이터 처리 성능을 최적화하여 빠른 응답 속도를 유지합니다. Supabase를 기반으로 한 안정적인 데이터 저장 및 인증 시스템 구축을 담당하고 있습니다.

담당 업무

  • 데이터베이스 스키마 설계 및 최적화
  • 백엔드 비즈니스 로직 구현
  • Supabase 연동 및 인증 시스템 구축
  • 데이터 파이프라인 및 쿼리 성능 최적화

기술 스택
