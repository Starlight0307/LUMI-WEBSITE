자주 묻는 질문

LUMI 사용 중 궁금한 점들을 정리했습니다.

LUMI는 인터넷 연결 없이도 사용 가능한가요?

네. LUMI는 완전한 오프라인 환경에서 동작하도록 설계되었습니다. 최초 설치 및 모델 다운로드 이후에는 인터넷 연결 없이도 모든 AI 기능을 사용할 수 있습니다.

어떤 GPU가 필요한가요?

NVIDIA RTX 3060 이상(VRAM 6GB+) 또는 Apple M 시리즈 칩셋을 권장합니다. GPU가 없는 환경에서도 CPU 모드로 제한적 동작이 가능합니다.

Windows 외 OS도 지원하나요?

Windows 10/11, macOS(Apple Silicon M1/M2/M3), Ubuntu 20.04 LTS 이상을 지원합니다.

필요한 최소 사양이 어떻게 되나요?

RAM 16GB 이상, VRAM 6GB 이상의 GPU, SSD 여유 공간 10GB 이상을 권장합니다. 자세한 사양은 다운로드 페이지에서 확인하세요.

어떤 AI 모델을 사용하나요?

Ollama 기반으로 Llama, Mistral, Gemma 등 다양한 오픈소스 LLM을 지원합니다. 사용자가 원하는 모델을 자유롭게 선택하여 로컬에서 구동할 수 있습니다.

IoT 기기 연동은 어떻게 하나요?

스마트홈 허브(Home Assistant 등)와 연동하거나 직접 API를 통해 IoT 기기를 등록할 수 있습니다. GitHub 저장소의 Wiki를 참고해 주세요.

개인 데이터는 어떻게 보호되나요?

모든 데이터는 사용자의 로컬 기기에만 저장되며 외부 서버로 전송되지 않습니다.

오픈소스인데 보안이 괜찮나요?

오픈소스는 코드가 공개되어 보안 취약점이 빠르게 발견·수정됩니다. GitHub에서 직접 코드를 확인하실 수 있습니다.

오픈소스인가요? 무료로 사용할 수 있나요?

네. LUMI는 완전한 오픈소스 프로젝트로 무료로 사용할 수 있습니다.

버그나 기능 요청은 어디에 남기나요?

GitHub Issues 또는 고객센터 폼을 통해 남겨주세요. 적극적으로 검토하여 반영하겠습니다.
