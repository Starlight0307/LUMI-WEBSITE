Local Engine AI Assistant

클라우드를 거치지 않는
강력한 나만의 로컬 AI 비서

LUMI는 외부 서버나 크롬 확장 프로그램 환경의 제약에서 벗어나, 당신의 로컬 하드웨어 인프라 상에서 직접 인퍼런스를 수행하고 작동하는 완전한 독립형 AI 어시스턴트입니다.

🔒

완벽한 데이터 오프라인 보안

로컬 머신 내부에서 모든 연산이 진행되므로, 기업 기밀 코드나 개인 자산 데이터가 외부 네트워크로 유출될 염려가 전혀 없습니다.

🚀

하드웨어 가속 기반 초고속 연산

로컬 GPU(CUDA / Metal) 가속 엔진을 직접 바인딩하여, 클라우드 트래픽 정체 상황과 무관하게 언제나 균일하고 압도적인 응답 속도를 보장합니다.

🛠️

네이티브 OS 통제 및 확장성

브라우저 샌드박스 환경에 갇힌 확장 프로그램과 달리, 로컬 파일 시스템 제어, 개발 툴 체인 연동 등 운영체제 수준의 깊은 자동화를 지원합니다.

설치 파일 다운로드

컴파일이 완료된 각 OS별 네이티브 독립 실행 파일을 다운로드하여 복잡한 빌드 과정 없이 즉시 LUMI AI를 사용할 수 있습니다.

🪟
Windows
Windows 10 / 11 (64-bit)
🍏
macOS
Apple Silicon (M1/M2/M3)
🐧
Linux
Ubuntu 20.04 LTS 이상

로컬 구동 최소 시스템 요구 사양

정적 메모리 (RAM) 최소 16GB 이상 권장 (LLM 로컬 적재 목적)
그래픽 카드 (GPU) NVIDIA RTX 3060 이상 (VRAM 6GB 이상 필수) 또는 Apple M 시리즈 칩셋
스토리지 가용 공간 최소 10GB 이상의 SSD 여유 공간 (로컬 모델 가중치 보관용)

Git에서 다운로드 및 빌드

오픈소스 저장소에서 소스코드를 직접 클론하여 커스텀 수정 사항을 반영하거나 직접 개발 환경 상에서 로컬 엔진을 구동할 수 있습니다.

Official GitHub Repository
github.com/lumi-ai-project/lumi-core
STEP 1 원격 저장소 클론 (Clone)

터미널 또는 커맨드라인 인터페이스를 열고 아래의 명령어를 입력하여 로컬 컴퓨터로 소스코드를 복제합니다.

git clone https://github.com/lumi-ai-project/lumi-core.git
cd lumi-core
STEP 2 로컬 개발 종속성 설치

프로젝트 구동 및 엔진 컴파일에 필요한 핵심 모듈과 의존성 패키지들을 빌드 툴을 이용하여 구성합니다.

npm install # 환경 및 구조에 맞게 yarn 혹은 pip 명령어 등으로 대체 가능
STEP 3 로컬 엔진 초기화 및 런타임 실행

로컬 하드웨어 런타임 환경 변수를 할당한 후, 소스코드를 빌드하고 프로그램을 직접 로컬 백그라운드 환경에서 구동합니다.

npm run dev:local
